# IoT Based Weather Monitoring System

## Overview
A basic temperature and humidity monitoring system using a microcontroller and sensor.

## Components Used
- Arduino / ESP8266
- DHT11 or DHT22 Sensor
- Jumper Wires
- Breadboard

## Features
- Temperature monitoring
- Humidity monitoring
- Real-time sensor data reading
- Serial monitor output

## Skills Used
- IoT Basics
- Sensor Interfacing
- Embedded C
- Arduino Programming
